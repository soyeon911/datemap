# Roadmap

## Phase 1: Date Map MVP

목표: 데이트 장소를 지도에 저장하고, 사진과 메모가 포함된 카드를 남긴다.

주요 기능:

- React Native + Expo 앱 프로젝트 생성
- SQLite 로컬 DB 구성
- 네이버 지도 표시
- 현재 위치 또는 지도 이동
- 장소 검색 또는 지도에서 위치 선택
- 장소 검색 프록시 연동 지점
- 검색 결과 선택 시 장소명, 주소, 좌표 자동 입력
- 데이트 카드 생성
- 장소별 사진 여러 장 첨부
- 장소별 한 줄 일기 입력
- 장소별 해시태그 입력
- 데이트 날짜 선택
- 같은 날짜에 여러 장소 저장
- 저장된 장소 마커 표시
- 날짜별 장소 목록 보기
- 월별 장소 필터 준비
- 네이버 지도에 방문 장소 위도/경도 기반 하트 마커 표시
- 저장된 장소 카드 가로 목록 표시
- 저장 카드에 대표 사진, 등록 날짜, 장소 이름, 한 줄 일기, 해시태그 표시
- 하단 저장 카드 자동 스와이프
- 드롭다운 날짜 프리셋 필터
- 모달형 DatePicker 기간/주차 필터
- 저장 기록이 있는 날짜 활성 표시
- 장소 추가 팝업
- 사진 보관함 권한 요청
- 선택 사진 앱 내부 문서 폴더 복사
- 카드 선택 시 상세 팝업
- 상세 팝업 우상단 수정/삭제 액션 배치
- 저장된 장소 수정
- 저장된 장소 삭제
- API 키와 권한 요청 보안 기준 적용

완료 기준:

- 앱에서 장소를 저장할 수 있다.
- 같은 날짜에 여러 장소를 저장할 수 있다.
- 같은 장소에 여러 사진을 연결할 수 있다.
- 저장된 기록을 네이버 지도 위 하트 마커로 볼 수 있다.
- 저장된 기록을 날짜 또는 기간별 카드 목록으로 볼 수 있다.
- 저장 기록이 있는 날짜를 DatePicker에서 확인할 수 있다.
- 장소 추가는 메인 화면 위의 팝업에서 처리된다.
- 선택한 사진은 앱 내부 저장소로 복사되어 대표 사진과 사진 개수를 확인할 수 있다.
- 카드 선택 시 상세 화면에서 삭제할 수 있다.
- 민감 설정이 저장소에 커밋되지 않는다.
- 위치와 사진 권한은 필요한 시점에만 요청된다.

## Phase 2: Search and Filters

목표: 저장한 데이트 기록을 조건별로 빠르게 찾는다.

주요 기능:

- 장소명 검색
- 메모 내용 검색
- 키워드 검색
- 주별 필터
- 월별 필터
- 카테고리 필터
- 최근 방문 장소 정렬
- 네이버 장소 검색 프록시 서버 구현
- 네이버 장소 검색 API 실제 연결

## Phase 3: Detail and Memory Layer

목표: 단순 저장을 넘어 추억을 풍부하게 관리한다.

주요 기능:

- 데이트 상세 화면
- 여러 장 사진 첨부
- 감정/분위기 태그
- 즐겨찾기
- 재방문 표시
- 장소별 방문 히스토리

## Phase 4: Account and Sync

목표: 사용자의 기록을 안전하게 보관하고 기기 간 동기화한다.

주요 기능:

- 로그인
- 클라우드 저장
- 사진 업로드 스토리지
- 커플 공유 공간
- 백업과 복구
- 사용자별 접근 권한 검증
- 커플 초대와 공유 권한 관리

## Phase 5: Recommendations

목표: 기록을 기반으로 다음 데이트를 돕는다.

주요 기능:

- 자주 가는 지역 분석
- 안 가본 카테고리 추천
- 기념일 주변 장소 추천
- 저장한 장소 기반 후보 리스트
