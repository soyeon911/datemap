# Decision 0001: App-first Date Map MVP

## 상태

Accepted

## 날짜

2026-06-26

## 결정

DateMap은 웹이 아니라 모바일 앱 방식으로 시작합니다. 첫 번째 구현 범위는 데이트 지도 기능으로 제한합니다.

## 이유

- 핵심 경험이 지도, 현재 위치, 사진 선택, 장소 저장에 가깝다.
- 사용자는 데이트 직후 모바일에서 빠르게 기록할 가능성이 높다.
- 사진과 위치 권한은 모바일 앱에서 자연스럽게 처리할 수 있다.
- 검색, 월별 회고, 커플 공유는 기본 데이터가 쌓인 뒤 붙이는 편이 적합하다.

## 포함 범위

- 네이버 지도 표시
- 장소 선택
- 데이트 카드 작성
- 장소별 사진 여러 장 첨부
- 메모 작성
- 지도 마커 표시
- 월별 카드 목록 기본 구조

## 제외 범위

- 계정/로그인
- 커플 공유
- 추천
- 복잡한 검색
- 서버 동기화

## 다음 결정 필요

- 사진 저장 방식
