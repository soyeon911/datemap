# Search and Timeline

이 문서는 MVP 이후 추가할 검색과 기간별 보기 기능을 정리합니다.

## 목표

사용자가 저장한 데이트 기록을 "언제", "어디", "무엇을 했는지" 기준으로 빠르게 찾게 합니다.

## 검색 기준

### 날짜 기준

- 주별 검색
- 월별 검색
- 특정 날짜 검색
- 기간 범위 검색
- 같은 날짜에 방문한 여러 장소 보기

### 장소 기준

- 장소명 검색
- 주소 검색
- 지역명 검색
- 지도 반경 검색

### 키워드 기준

- 카페
- 식당
- 전시
- 영화
- 산책
- 여행
- 기념일
- 직접 만든 태그
- 한 줄 일기에 포함된 해시태그

### 내용 기준

- 한 줄 일기 검색
- 사진 설명 검색은 후순위

## 화면 계획

### 검색 화면

- 상단 검색 입력
- 기간 필터
- 키워드 칩
- 결과 카드 목록
- 지도에서 결과 보기 전환

### 월별 타임라인

- 월 단위 섹션
- 해당 월의 데이트 날짜
- 날짜별 장소 목록
- 방문 지역 요약
- 해시태그 요약

### 주별 보기

- 한 주의 데이트 기록
- 주말/평일 구분
- 캘린더와 연결 가능한 구조

## 데이터 준비

MVP 단계부터 아래 필드를 저장해야 검색 기능을 쉽게 붙일 수 있습니다.

- `date`
- `year`
- `month`
- `weekOfYear`
- `placeName`
- `address`
- `hashtags`
- `oneLineDiary`
- `normalizedSearchText`

## 구현 우선순위

1. 장소명 검색
2. 날짜별 장소 보기
3. 월별 필터
4. 주별 필터
5. 해시태그 필터
6. 한 줄 일기 검색
6. 지도 반경 검색
